# Roadmap: Генератор жалоб на УО «Континент Комфорта»
## г. Химки, ул. Совхозная, д. 5

---

## Контекст проекта

УО «Континент Комфорта» прекратила обслуживание дома после смены руководства в декабре. Совет дома принял решение о смене УО. Для досрочного расторжения договора нужны массовые жалобы от жильцов (20-30 в день) в течение 2-3 недель.

**Цель сервиса:** максимально снизить порог входа для подачи жалоб. Жилец открывает страницу → выбирает проблемы → получает готовый текст → отправляет в несколько инстанций за 2-5 минут.

**Текущее состояние:** работающий HTML-генератор с выбором проблем, двумя тонами (формальный/неформальный), фото, QR-листовка для лифта. Отправка только в УО по email.

**Репозиторий:** https://github.com/catch-the-wave/KhimkiManagement.git

## Архитектура UI: роутер жалоб

### Концепция

Вместо списка каналов — **пошаговый мастер из 2-3 вопросов**, который выдаёт персональный план действий. Всё должно работать на телефоне, крупные кнопки, минимум текста.

### Дерево решений

```
Текст жалобы сгенерирован
    │
    ├─ Вопрос: «Есть подтверждённый аккаунт Госуслуг?»
    │      │
    │      ├─ НЕТ / НЕ ЗНАЮ
    │      │    │
    │      │    └─ Показать email-каналы (7 кнопок mailto:)
    │      │       + телефон главы Химок
    │      │       + предложение «Подключите Госуслуги через Сбер/Тинькофф за 5 мин»
    │      │
    │      └─ ДА
    │           │
    │           ├─ Показать ВСЁ:
    │           │    ├─ Email-каналы (те же 7 кнопок) — «обязательный минимум»
    │           │    ├─ Добродел (скопировать текст + открыть портал)
    │           │    ├─ ЕДС МО (скопировать текст + открыть портал)
    │           │    ├─ ГИС ЖКХ (скопировать текст + открыть deep-link)
    │           │    └─ Если выбраны крысы/тараканы → Роспотребнадзор онлайн
    │           │
    │           └─ Подвопрос: «Установлено приложение Добродел?»
    │                ├─ ДА → инструкция «откройте → вставьте текст»
    │                ├─ НЕТ → ссылки на AppStore/GooglePlay/RuStore
    │                └─ ЛУЧШЕ ЧЕРЕЗ САЙТ → инструкция для vmeste.mosreg.ru
    │
    └─ После каждой отправки:
           → Галочка ✅ на кнопке
           → Счётчик «Отправлено: 3 из 7»
           → Готовый текст для копирования в чат дома
```

### Режимы по времени

| Режим | Время | Что делать | Каналы |
|---|---|---|---|
| ⚡ «5 минут» | 5 мин | Обязательный минимум для каждого | Email: УО + ЖКХ Химок + ГЖИ МО |
| 📱 «15 минут» | 15 мин | + порталы через Госуслуги | + Добродел + ЕДС МО |
| 🔨 «Тяжёлая артиллерия» | 20 мин | Эскалация (через 1-2 недели) | + Прокуратура + Роспотребнадзор + Губернатор |

### Режим «по одной проблеме» для Добродела

На Доброделе одна жалоба = одна проблема. Если жилец выбрал 5 проблем, для Добродела нужно подать 5 отдельных жалоб. Генератор должен уметь:
- выдавать текст по одной проблеме за раз
- показывать прогресс: «Проблема 1 из 5: Уборка подъезда → [Скопировать] → [Открыть Добродел]»
- после отправки → «Проблема 2 из 5: Мусоропровод → [Скопировать]...»

20 жильцов × 5 проблем = 100 обращений. Каждое требует отдельного ответа за 8 дней.

## Два типа текста жалобы

### Тип 1: Для УО и общих каналов (уже реализовано)

Описание проблем + требование устранить. Два тона: формальный и неформальный. Случайная вариативность текста (seed-based).

### Тип 2: Для ГЖИ / ГИС ЖКХ / прокуратуры (НОВОЕ)

Отличия от типа 1:
- Акцент на неисполнение обязанностей УО (а не на саму проблему)
- Ссылки на предыдущие безответные обращения в УО
- Юридические формулировки:
  - «Прошу провести проверку деятельности УО "Континент Комфорта"»
  - «Прошу выдать предписание об устранении нарушений»
  - «Прошу рассмотреть вопрос о досрочном расторжении договора управления»
  - «Прошу привлечь к административной ответственности по ст. 7.22 КоАП РФ»

Генератор автоматически выбирает нужный тип в зависимости от адресата.

## Трекинг массовости: шаблон для чата дома

После каждой отправки сервис предлагает скопировать отчёт в чат дома:

```
✅ 22.03.2026 | Email в УО | kontinent.reception@mail.ru | Тема: уборка, мусоропровод, крысы
✅ 22.03.2026 | Email в ЖКХ Химок | zhkh@admhimki.ru | Тема: уборка, мусоропровод, крысы
✅ 22.03.2026 | Добродел | №12345 | Тема: уборка подъезда
```

Это даёт совету дома:
- Публичный учёт кто подал и куда
- Массив доказательств массовости
- Мотивацию для других жильцов
- Материал для прокуратуры/суда

## Встроенные инструкции для порталов

Для каждого канала с Госуслугами — мини-инструкция внутри генератора (не отдельный файл). Формат: 5 шагов, вертикальные карточки, крупный текст, без таблиц. Адаптировано под мобильный экран.

### Добродел (5 шагов)

```
Шаг 1: Откройте vmeste.mosreg.ru → «Войти»
Шаг 2: Войдите через Госуслуги
Шаг 3: «Сообщить о проблеме» → «Дом, ЖКХ»
Шаг 4: Адрес: Химки, Совхозная, 5
Шаг 5: Вставьте текст [📋] → фото → Отправить
```

### ЕДС МО (4 шага)

```
Шаг 1: Откройте eds.mosreg.ru/claims
Шаг 2: Введите адрес: Химки, Совхозная, 5
Шаг 3: Выберите категорию (уборка / мусоропровод / освещение...)
Шаг 4: Вставьте текст [📋] → Отправить
```

### ГИС ЖКХ (5 шагов)

```
Шаг 1: Откройте dom.gosuslugi.ru → войдите
Шаг 2: «Направить обращение или жалобу»
Шаг 3: Адрес: МО → Химки → Совхозная → 5
Шаг 4: Адресат: «ГЖИ МО» (для максимальной силы)
Шаг 5: Вставьте текст [📋] → фото → Отправить
```

## Технические заметки

### Стек
- Чистый HTML/CSS/JS, без фреймворков
- Self-contained файл (один .html)
- Оптимизирован под мобильный (viewport, touch, safe-area)
- Работает оффлайн (кроме QR-генерации и mailto)

### Хостинг
- GitHub Pages: github.com/catch-the-wave/KhimkiManagement.git
- Альтернативы: Netlify Drop, Telegram-бот @AnyFileBot

### Ограничения mailto
- Большинство браузеров/приложений поддерживают mailto до ~28000 символов
- Если текст длиннее — fallback: копировать в буфер + открыть пустой mailto
- На iOS: mailto открывает стандартное приложение Mail (или Gmail если настроено)
- На Android: выбор приложения из списка

### QR-код
- Используется api.qrserver.com для генерации
- Флаер для лифта — отдельный файл flyer-elevator.html
- QR ведёт на захостированный complaint-generator.html

## Ключевые инсайты из исследования

1. **ЕСИА — главный барьер.** Без Госуслуг жильцы не могут пользоваться ни одним госпорталом. Email — единственный путь для ~30-50% жильцов (пожилые, арендаторы, технически неуверенные).

2. **Добродел — самый мощный канал давления.** Не потому что юридически сильнее, а потому что муниципалитеты рейтингуются по % решённых проблем. Кнопка «Не решено» возвращает жалобу на спецконтроль. Это создаёт реальную мотивацию для чиновников.

3. **ГИС ЖКХ — максимальная юридическая сила.** Жалобы здесь = документальные доказательства для суда. Deep-link на форму работает: `https://my.dom.gosuslugi.ru/citizen-cabinet/#!/appeal/card/create/?isApplicant=true`

4. **Одна проблема = одна жалоба (на Доброделе).** 20 жильцов × 5 проблем = 100 обращений. Каждое требует отдельного ответа за 8 дней.

5. **Обязательно сначала в УО.** Все надзорные органы требуют доказательства предварительного обращения в УО. Email на kontinent.reception@mail.ru — это обязательный первый шаг. Без него ГЖИ вправе не начинать проверку.

6. **«Госуслуги.Дом» — коллективные заявки.** С сентября 2024 приложение поддерживает коллективные жалобы, где соседи присоединяются к одному обращению.

7. **Email — самый масштабируемый канал.** 7 адресов × 1 нажатие = жалоба в 7 инстанций за 2 минуты. Никакой регистрации. Работает с любого телефона.
